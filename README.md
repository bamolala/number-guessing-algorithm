Start Program

1. Generate random number between 1 and 10.
2. Start loop for correct guesses.
3. Prompt user to guess a number.
4.  While guess is not random number, Do
 if( guess = random number){
    display congratulatory message to user and end program.
}elseif( guess > random number){
    tell user guess is too high, try again.
}else( guess < random number>){
    tell user guess is too low, try again.
}
end while loop

End Program
